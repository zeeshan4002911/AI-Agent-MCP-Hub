# API Backend app
This app is built on top of FastAPI

# for running the api-backend
uvicorn main:app --reload

## Test the Interactive API Docs
FastAPI dynamically parses your type hints to build interactive, self-generating web documentation.

Swagger UI Docs: Open your web browser and navigate to http://127.0.0.1:8000/docs. This page presents an interactive interface where you can test your POST and GET requests live right from the screen.

ReDoc Docs: Alternatively, you can view beautifully structured, read-only documentation at 
http://127.0.0.1:8000/redoc
